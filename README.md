# Code Challenge - BE Engineer
## Description
This repository contains the solution for the Backend Engineer code challenge for 99tech Company.

This challenge focuses on 3 main problems:
1. Problem 4: Implement 3 ways to sum numbers from 1 to N.
2. Problem 5: Design a simple CRUD API for managing a resource
3. Problem 6: Provide a technical design for a real-time score board system.

## Structure
- `src/problem4/`: Contains the solution for Problem 4. Go to the this [README.md](./src/problem4/README.md) file inside for more details.
- `src/problem5/`: Contains the solution for Problem 5. Go to the this [README.md](./src/problem5/README.md) file inside for more details.
- `src/problem6/`: Contains the solution for Problem 6. Go to the this [README.md](./src/problem6/README.md) file inside for more details.

## Technologies Used
- Node.js
- Express.js
- TypeScript
- SQLite
- Sequelize ORM
